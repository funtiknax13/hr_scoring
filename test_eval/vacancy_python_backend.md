# Вакансия: Python Backend Developer (Middle/Senior)

## О компании
Финтех-компания, разрабатывающая платформу для управления корпоративными расходами. 200+ сотрудников, продукт работает в 15 странах.

## Задачи
- Проектировать и разрабатывать REST API и микросервисы на Python
- Работать с высоконагруженными системами (десятки миллионов транзакций в месяц)
- Проводить код-ревью, участвовать в архитектурных обсуждениях
- Оптимизировать запросы к PostgreSQL, работать с индексами и планами запросов
- Интегрировать внешние платёжные системы и банковские API
- Писать тесты (unit, integration), поддерживать покрытие на уровне 80%+

## Требования
- Опыт коммерческой разработки на Python от 3 лет
- Уверенное знание FastAPI или Django/DRF
- PostgreSQL: сложные запросы, оптимизация, миграции (Alembic или аналоги)
- Celery + Redis: опыт построения очередей задач
- Понимание принципов SOLID, чистой архитектуры, паттернов проектирования
- Опыт работы с Docker, базовые знания Kubernetes будут плюсом
- Умение читать и писать на английском (документация, переписка с партнёрами)

## Будет плюсом
- Опыт в финтехе или работы с платёжными системами
- Знание asyncio, опыт с asyncpg или aiohttp
- Участие в open-source проектах
- Опыт настройки мониторинга (Prometheus, Grafana, Sentry)
